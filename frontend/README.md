# Buraki Frontend

Next.js 15 + React 19 + TypeScript ile geliştirilmiş web arayüzü. Harita için MapLibre GL, sunucu durumu için TanStack Query, grafikler için Recharts kullanılır.

## Komutlar

```bash
npm install
npm run dev        # http://localhost:3000
npm run build
npm run lint
```

`.env.local` içeriği:

```
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000/api/v1
```
